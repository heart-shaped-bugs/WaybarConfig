# Waybar Configuration

Моя персональная конфигурация статус-бара **Waybar** для Wayland композиторов (Hyprland, Sway, River и др.).

### Используемые модули
- Shutdown (Выключение)
- AppCenter (Системный трей)
- Media Player (через `playerctl`)
- CurrentTab (Текущая вкладка)
- Workspaces (Гиперпространства)
- Network (Сеть)
- PulseAudio/Pipewire (Звук)
- Brightness (Яркость)
- Battery (Батарея)
- Clock (Дата и время)
